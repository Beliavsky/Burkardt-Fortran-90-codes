knapsack_values_test

knapsack_values_test, a Fortran90 code which calls knapsack_value(), which returns some simple examples of the knapsack problem, in which N items are available, each with given value V and weight W. A thief's knapsack can carry no more than K pounds. The thief seeks a selection S of items to carry in the knapsack of maximum total value.

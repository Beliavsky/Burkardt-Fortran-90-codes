channel, a Fortran90 code which computes the time-independent flow of a viscous incompressible fluid in a 2D channel.

Source Code:

  • channel_test.sh, runs all the tests.
  • channel_test.txt, the output file.

The program writes out "XY" (node coordinate) and "UV" (velocity component) data files for the optimized solution.

  • xy.txt, grid information.
  • uv.txt, velocity information.
  • uv_dir.png, a png image of the velocity direction field.
  • uv_vec.png, a png image of the velocity field.

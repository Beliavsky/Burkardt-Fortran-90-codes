sparse_count

sparse_count, a Fortran90 code which contains routines for the analysis and construction of sparse grids in which a fixed family of 1D quadrature rules is used for all spatial dimensions.

Languages:

sparse_count is available in a C++ version and a Fortran90 version and a MATLAB version and an Octave version.

quadrature_weights_vandermonde_test

quadrature_weights_vandermonde_test, a Fortran90 code which calls quadrature_weights_vandermonde(), which illustrates a method for computing the weights of a quadrature rule, assuming that the points have been specified, by setting up a linear system involving the Vandermonde matrix.

quadrature_weights_vandermonde_2d_test

quadrature_weights_vandermonde_2d_test, a FORTRAN90 code which calls quadrature_weights_vandermonde_2d(), which illustrates a method for computing the weights W of a 2D interpolatory quadrature rule, assuming that the points (X,Y) have been specified, by setting up a linear system involving the Vandermonde matrix.

cube_monte_carlo

cube_monte_carlo, a FORTRAN90 code which estimates the integral of a function over the interior of the unit cube in 3D.

Languages:

cube_monte_carlo is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

two_body_ode_test

two_body_ode_test, a Fortran90 code which calls two_body_ode(), which simulates a planar two body problem, writing graphics files for processing by gnuplot().

two_body_ode, a Fortran90 code which simulates a planar two body problem, writing graphics files for processing by gnuplot().

Source Code:

INITIAL_ORBIT starts the problem, and runs it for just two orbits.

  • initial_orbit.sh, runs all the tests.
  • initial_orbit.txt, the output file.
  • initial_orbit_commands.txt, input commands to GNUPLOT.
  • initial_orbit_data.txt, data for GNUPLOT.
  • initial_orbit.png, a plot of the data.

ORBITAL_DECAY runs the problem for 20 orbits.

  • orbital_decay.sh, runs all the tests.
  • orbital_decay.txt, the output file.
  • orbital_decay_commands.txt, input commands to GNUPLOT.
  • orbital_decay_data.txt, data for GNUPLOT.
  • orbital_decay.png, a plot of the data.

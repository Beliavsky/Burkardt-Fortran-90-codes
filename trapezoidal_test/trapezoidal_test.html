trapezoidal_test

trapezoidal_test, a Fortran90 code which calls trapezoidal(), which solves one or more ordinary differential equations (ODE) using the (implicit) trapezoidal method, with a version of fsolve() handling the associated nonlinear equation, and using a fixed time step.

tetrahedron_jaskowiec_rule_test

tetrahedron_jaskowiec_rule_test, a Fortran90 code which calls tetrahedron_jaskowiec_rule(), which returns quadrature rules, with exactness up to total degree 20, over the interior of a tetrahedron in 3D, by Jan Jaskowiec, Natarajan Sukumar.

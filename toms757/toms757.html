toms757

toms757, a FORTRAN90 code which evaluates a number of uncommon special functions, by Allan McLeod.

The code is a FORTRAN90 version of ACM TOMS algorithm 757.

The special functions include:

  • the Abramowitz function of several orders;
  • integrals of the Airy functions Ai(x) and Bi(x);
  • Clausen's integral;
  • Debye integrals of several orders;
  • integrals of the Bessel functions I0, J0, K0 and Y0;
  • Struve functions;
  • the synchrotron radiation integral;
  • the transport integral of several orders;

The version displayed here has been converted to FORTRAN90, and other internal changes have been made to suit me.

The text of many ACM TOMS algorithms is available online through ACM: https://calgo.acm.org/ or NETLIB: https://www.netlib.org/toms/index.html.

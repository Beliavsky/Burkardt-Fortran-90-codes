test_matrix_test

test_matrix_test, a Fortran90 code which calls test_matrix() to define test matrices for which some of the determinant, eigenvalues, inverse, null vectors, P*L*U factorization or linear system solution are already known, including the Vandermonde and Wathen matrix.

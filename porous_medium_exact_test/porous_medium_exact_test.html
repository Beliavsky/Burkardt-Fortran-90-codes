porous_medium_exact_test

porous_medium_exact_test, a Fortran90 code which calls porous_medium_exact(), which returns an exact solution of the porous medium equation (PME), dudt=Del^2(u^m), a partial differential equation (PDE) related to the diffusion equation, based on the Barenblatt solution.

tec_to_fem_test

tec_to_fem_test, a Fortran90 code which calls tec_to_fem(), which reads a TECPLOT file defining a finite element dataset, and writes a corresponding set of FEM files.

Source Code:

  • tec_to_fem_test.sh, runs all the tests.
  • tec_to_fem_test.txt, the output file.

TINY is a "tiny" dataset.

  • tiny.dat, a TECPLOT ASCII file defining the dataset.
  • tiny.nodes.txt, the FEM file containing the node coordinates.
  • tiny.elements.txt, the FEM file containing the elements.
  • tiny.values.txt, the FEM file containing the node data.

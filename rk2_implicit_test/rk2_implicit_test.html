rk2_implicit_test

rk2_implicit_test, a Fortran90 code which calls rk2_implicit(), which solves one or more ordinary differential equations (ODE) using a Runge-Kutta order 2 implicit method, with a version of fsolve() handling the associated nonlinear equation, and using a fixed time step.

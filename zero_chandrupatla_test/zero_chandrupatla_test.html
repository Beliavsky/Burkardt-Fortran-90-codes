zero_chandrupatla_test

zero_chandrupatla_test, a Fortran90 code which calls zero_chandrupatla(), which finds a zero of a scalar function of a scalar variable, starting from a change of sign interval, using the Chandrupatla method, which can converge faster than bisection, regula falsi, or Brent's method, by Tirupathi Chandrapatla.

starpac_test

starpac_test a Fortran90 code which calls starpac(), which performs statistical data analysis. It includes functions for nonlinear least squares regression, time series analysis in both time and frequency domains), line printer graphics, basic statistical analysis, and linear least squares regression, by Janet Donaldson.

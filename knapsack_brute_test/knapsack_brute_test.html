knapsack_brute_test

knapsack_brute_test, a Fortran90 code which calls knapsack_brute(),which uses brute force to solve small versions of the knapsack problem, in which N items are available, each with given value and weight. A thief's knapsack can carry no more than Wmax pounds. The thief seeks a selection of items to carry in the knapsack of maximum total value.

hypergeometric_test

hypergeometric_test, a Fortran90 code which calls hypergeometric(), which evaluates the generalized hypergeometric functions pFq, for integer p and q, for complex arguments of large magnitude, by Warren Perger, Atul Bhalla, and Mark Nardin.

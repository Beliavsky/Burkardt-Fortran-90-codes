midpoint_adaptive_test

midpoint_adaptive_test, a Fortran90 code which calls midpoint_adaptive(), which solves one or more ordinary differential equations (ODE) using the (implicit) midpoint method, relying on fsolve() to solve the implicit equation, and using an adaptive timestep.

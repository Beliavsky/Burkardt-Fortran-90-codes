sigmoid_derivative_test

sigmoid_derivative_test, a Fortran90 code which calls sigmoid_derivative(), which evaluates derivatives of any order for the sigmoid function s(x)=1/(1+exp(-x)). The test code creates graphics images using gnuplot().
